##Tic Tac Toe Game

##Description
A console-based Tic Tac Toe game developed using Java that allows two players to play alternately on a 3Ã—3 grid until a player wins or the game ends in a draw.

##Technologies Used
-java
-Eclipse
-GitHub

##Features
-Two-player gameplay (X and O)
-Console-based user interaction
-Input validation for player moves
-Automatic win and draw detection
-Simple and clean game logic

##How to Run
1.Clone the repository
2.Import the project into your IDE
3.Open the main class
4.Run the project as a Java Application
